## What is a feature flag (toggle, flipper)? A simple analogy.

## The core problem they solve: Decoupling deployment from release.

## Quick examples of where you've encountered them (even if you didn't know it).

- A new UI element appearing for some users.

- A/B testing different headlines on a news site.

- A "beta features" section in an app.

## Why this is a system design concern: It's not just an if statement.
